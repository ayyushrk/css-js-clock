# CSS JS Clock ⏰

An analog clock built with pure HTML, CSS, and JavaScript — no libraries, no frameworks.

Built as part of [JavaScript30](https://javascript30.com/) by [Wes Bos](https://wesbos.com/).

## Live Demo

🔗 [ayyushrk.github.io/css-js-clock](https://ayyushrk.github.io/css-js-clock/)

## What I Learned

- CSS `transform-origin` to rotate hands from the correct anchor point
- How to calculate degrees for hour, minute, and second hands
- Fixing the second hand backward flick at :00 using `transition: none`
- Smooth hour hand movement by factoring in minutes

## Built With

- HTML
- CSS
- Vanilla JavaScript
